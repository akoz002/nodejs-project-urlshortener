API Project: URL Shortener Microservice

User Stories:

  1. I can post a URL to the following link:

    POST [project_url]/api/shorturl/new

    and I will receive a shortened URL in the JSON response. For example:

    { "original_url": "https://www.google.com/", "short_url": 1 }

  2. If I pass an invalid URL that doesn't follow the format:

    http(s)://www.example.com(/more/routes)

    the JSON response will contain an error like { "error": "invalid URL" }. To be sure that the submitted url points to a valid site use the function dns.lookup(host, cb) from the dns core module.
  3. When I visit the following link, it will redirect me to my original link:

    GET [project_url]/api/shorturl/[short_url]

Example Usage:

Create Short URL:

POST [project_url]/api/shorturl/new

body (urlencoded): 'url=https://www.google.com/'

Returns:

  • {
    • "original_url": "https://www.google.com/",
    • "short_url": 1
  • }
    •

Visit Short URL:

GET [project_url]/api/shorturl/1

Redirects to: https://www.google.com/
